# CRUD
CRUD em PHP, MySQL e Materialize CSS

## Sobre
Sistema com um simples **CRUD** (Create, Read, Update, Delete) feito em **PHP**, **MySQL** e **Materialize css 1.0 beta**. Código feito de forma procedural conectando ao banco de dados MySQL e com uma simples estilização visual responsiva.
